## 2. Critical

> 🚫 **发现 1 处 Critical 级别问题**

### [C-1] 版本要求与限定词遗漏
- **问题类型**：信息遗漏
- **位置**：`Configuration Requirements > Basic Settings` 第 2 段
- **原文**：
  ```rst
  This configuration is only available in CodeIgniter 4.3 or later versions.
  You must set the ``$cachePath`` parameter before using the cache.
  ```
- **译文**：
  ```rst
  此配置在 CodeIgniter 4.3 版本可用。
  使用缓存前需要设置 ``$cachePath`` 参数。
  ```
- **问题**：遗漏了 `only` 与 `or later versions`，并将 `must` 弱化为“需要”。这会让读者误以为 4.3 是唯一适用版本，且把强制要求误读为一般建议。
- **建议**：
  ```rst
  此配置仅在 CodeIgniter 4.3 或更高版本中可用。
  使用缓存前必须设置 ``$cachePath`` 参数。
  ```
- **影响**：会误导版本支持范围和配置前提，属于阻断定稿的问题。
- **复验建议**：修复后重新对照英文原文核查限定词，并重新运行 guard。

## 3. Major

> ⚠️ **发现 2 处 Major 级别问题**

### [M-1] `helper` 术语译法不一致
- **问题类型**：术语不一致
- **位置**：`Using Cache Helpers` 多处
- **原文**：
  ```rst
  Use the cache helper to store data.
  ```
- **译文**：
  ```rst
  使用缓存帮助函数存储数据。
  ```
- **问题**：项目内标准译法为“辅助函数”，当前译文使用“帮助函数”，与既有文档不一致。
- **建议**：
  ```rst
  使用缓存辅助函数存储数据。
  ```
- **影响**：降低术语一致性和专业性，建议发布前统一修复。
- **复验建议**：全文搜索 `帮助函数`，确认同文档内已统一为“辅助函数”。

### [M-2] 标题装饰线长度不足
- **问题类型**：格式破坏
- **位置**：`Configuration Options` 标题
- **原文**：
  ```rst
  Configuration Options
  ---------------------
  ```
- **译文**：
  ```rst
  配置选项
  -------
  ```
- **问题**：中文标题长度增加后，装饰线未同步调整，长度不足。
- **建议**：
  ```rst
  配置选项
  --------
  ```
- **影响**：可能触发 Sphinx 构建警告，属于发布前应修复的格式问题。
- **复验建议**：修复后重新运行 guard，并在本地执行 `make html` 做最终确认。

## 4. Minor

> 💡 **发现 1 处 Minor 级别问题**

### [m-1] 长句可拆分以提升可读性
- **位置**：`Cache Lifetime` 说明段
- **当前译文**：
  ```rst
  默认缓存生命周期为 60 秒，如果需要调整此值可以在配置文件中修改 ``$lifetime`` 参数，该参数接受整数秒数或使用时间表达式如 ``3600`` 表示一小时。
  ```
- **优化建议**：
  ```rst
  默认缓存生命周期为 60 秒。如需调整，可在配置文件中修改 ``$lifetime`` 参数。
  该参数接受整数秒数或时间表达式，如 ``3600`` 表示一小时。
  ```
- **理由**：拆分后信息层次更清晰，也更符合中文技术文档的行文节奏。
