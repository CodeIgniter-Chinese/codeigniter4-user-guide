总结
==========

本指南介绍了如何使用 CodeIgniter4 构建用于管理图书和作者的简单 RESTful API。内容涵盖搭建 CodeIgniter 项目、配置数据库、创建模型，以及通过控制器和自动路由实现 API 接口，并利用 ``ResponseTrait`` 确保 API 响应的一致性。

在此基础上，可通过添加更多资源来扩展 API，使用 `CodeIgniter Shield <https://shield.codeigniter.com/>`__ 实现身份验证与授权，并进一步探索 :doc:`../../libraries/throttler`、:doc:`../../incoming/controller_attributes` 和 :doc:`../../general/caching` 等高级特性。CodeIgniter 的灵活性与强大功能使其成为构建健壮 API 的绝佳选择。
