###########################
为 CodeIgniter 做贡献
###########################

CodeIgniter 是一个由社区驱动的项目，接受社区贡献的代码和文档。这些贡献以 GitHub 上的 `CodeIgniter4 仓库 <https://github.com/codeigniter4/CodeIgniter4>`_ 的 Issues 或 `Pull Requests <https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests>`_ 形式提交。

如果你希望做出贡献，请参阅我们代码仓库中的 `为 CodeIgniter4 做贡献 <https://github.com/codeigniter4/CodeIgniter4/tree/develop/contributing>`_ 部分。
