版本 4.0.0-beta.3

发布日期:2019 年 5 月 7 日

  • 亮点

  • 新消息

  • 应用变更

  • 变更的文件

  • 合并的 PR

亮点

  • 在整个项目中添加了类型提示并纠正了拼写错误(参见 API 文档)

  • 修复了许多 model、database、validation 和 debug toolbar 问题

新消息

  • Database.FieldNotExists

  • Validation.equals、not_equals

应用变更

  • 在 app/Config/App 中删除了 $salt 配置项

  • 在 app/Config/Migrations 中默认启用了迁移

  • 简化了 public/.htaccess
